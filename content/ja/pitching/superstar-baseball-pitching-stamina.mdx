## スーパースターベースボールの投球スタミナの核心的メカニズム

クッパのようなパワータイプからピーチのようなテクニックタイプまで、ゲーム内のすべてのキャラクターは、正確に**10ポイントのスタミナ**を持って試合を開始します。これはキャラクターのサイズや役割に関係なく共通のステータスです。投球ごとにエネルギーが減少する一般的な野球シミュレーションゲームとは異なり、このゲームでは主要なイベントに対してのみペナルティが課せられます。

投手がこれらのポイントを失う主な要因は2つあります。1つ目は、自責点を与えると即座に1ポイント減点されます。2つ目は、キャラクター固有の強力な必殺技である「スター投球」を投げると、1ポイントを消費します。これらのトリガーを理解することで、投手のパフォーマンスが低下し始めるまで、あとどれくらいの「寿命」が残っているかを正確に計算できるようになります。

| アクション | スタミナ減少値 | 投手への影響 |
| :--- | :--- | :--- |
| **スター投球を投げる** | -1 ポイント | リリース時に即座に減点。 |
| **自責点を与える** | -1 ポイント | ランナーがホームを踏んだ時点で減点。 |
| **非自責点を与える** | 0 ポイント | エラーや盗塁による失点にはペナルティなし。 |
| **通常の投球をする** | 0 ポイント | 通常の投球やチャージ投球による減点はなし。 |

> **💡 ヒント:** 序盤のイニングではスター投球を控えめに使いましょう。スター投球は失点と同じ量のスタミナを消費するため、乱用するとリードしていても投手が早く疲れてしまう原因になります。

## 「疲れ」状態：7失点（ポイント）のしきい値

ゲーム内で最も分かりやすい疲労のサインは、投手の横に表示される「汗（しずく）」アイコンです。このアイコンは、投手が**「疲れ」状態**に入ったことを示します。これは投手が**7ポイントのスタミナ**を失った（残り3ポイントになった）時点で発生します。

投手が疲れると、マウンドでの有用性は著しく低下します。最も壊滅的な影響は、変化球（カーブ）のコントロールがほぼ完全に失われることです。疲れた投手で変化球を投げようとしても、アナログスティックの入力に関係なく、ボールは直線的に進みます。わずかな変化を取り戻す唯一の方法は「チャージ投球」を使用することですが、これらは球速が大幅に遅くなり、打者にとってタイミングを合わせやすくなります。

| 残りスタミナ | 視覚的インジケーター | 投球能力 |
| :--- | :--- | :--- |
| **4 ～ 10 ポイント** | なし | 完全なコントロール、最大の変化、通常の球速。 |
| **1 ～ 3 ポイント** | **汗アイコン** | 通常投球で変化なし。チャージ投球で変化・球速が減少。 |
| **0 ポイント** | **汗アイコン** | 変化なし。**スター投球使用不可**。 |

## 完全な限界とスター投球の制限

7ポイントの時点でも投球は困難になりますが、10ポイントに達するとほぼ不可能になります。**0スタミナポイント**になると、投手は「限界（スタミナ切れ）」と見なされます。この状態では、スター投球を使用する能力が完全に無効化されます。チームのストックにスターアイコンがあっても、Rボタンを押しても通常の投球になります。

これは戦略的なジレンマを生みます。もし6失点しており、すでに3回スター投球を使っているなら、すでに9ポイントを消費しています。あともう1失点、あるいはもう1回必殺技を使うだけで、あなたの投手は「的」同然となり、遅くて真っ直ぐなボールしか投げられなくなります。

### 自責点 vs. 非自責点
このゲームには、自責点を追跡するための驚くほど洗練されたロジックが備わっています。外野手がフライを落とした（エラー）、あるいは盗塁によってランナーが生還した場合、**スーパースターベースボールの投球スタミナ**は通常維持されます。システムは主に、ヒットや四球など、投手の直接的な責任とされるプレーによる失点に対してペナルティを与えます。

| シナリオ | スタミナ減少？ | 理由 |
| :--- | :--- | :--- |
| **ソロホームラン** | **はい** | 投手が打たれた直接的なヒット。 |
| **エラーによる失点** | **いいえ** | 守備のミスであり、投球のミスではない。 |
| **押し出しによる失点** | **はい** | 投手がストライクゾーンを外した結果。 |
| **犠牲フライ** | **はい** | 失点は投手の記録として計上される。 |

## スタミナに関する都市伝説の検証

対戦コミュニティでは、特定のキャラクターに関して、いくつかの都市伝説が流布しています。よく信じられていたのは、看板キャラクターであるマリオとルイージは隠された11ポイント目のスタミナを持っている、あるいは疲労に耐性があるという説でした。しかし、厳密な検証によりこれが誤りであることが証明されています。ロースター内のすべてのキャラクターは、全く同じ10ポイント減少ルールに従います。

もう一つの誤解は、キャラクター紹介メニューにある高い「スタミナ」ステータスが、より多くの投球を可能にするというものです。『スーパーマリオスタジアム ミラクルベースボール』において、このスタミナステータスは実際にはキャラクターの「チャージ」メーターが溜まる速さや、最高速度を維持できる時間を指しており、「汗」アイコンへの耐性ではありません。キャラクター固有のステータスの詳細については、[Official Mario Wiki](https://www.mariowiki.com/Mario_Superstar_Baseball) で各属性の比較を確認できます。

## スーパースターベースボールの投球スタミナの管理

9イニングの試合を通じて高いレベルのプレーを維持するには、管理戦略を採用する必要があります。以下の手順に従うことで、限界状態のキャラクターで投球を強いられることを防げます。

1.  **スコアボードを監視する:** 自責点の数を頭の中で数えておきましょう。5失点に達したら、リリーフ投手の準備を考え始めてください。
2.  **スターを計算する:** すべてのスター投球は「自発的な」スタミナ喪失です。すでに4失点している場合は、予定より早く「疲れ」状態になるのを避けるため、スター投球の使用をやめましょう。
3.  **汗アイコンを見極める:** 汗アイコンが表示された瞬間、相手はあなたが変化球を投げられないことを察知します。イニング終了まであと1アウトという状況でない限り、すぐに投手を交代させてください。
4.  **「BAM」テクニックを活用する:** **スーパースターベースボールの投球スタミナ**が低くても、「バルキー・アングル・マニピュレーション（BAM）」（投球中に上下を押す）を使用することで、軌道をわずかに変えることができます。ただし、これは健康な投手の変化球の代わりとしては不十分です。

> **警告:** 限界状態のプレイヤーで対人戦を行うのは非常に危険です。変化球やスター投球が使えない状態では、熟練した打者は簡単にタイミングを予測し、長打を放ってくるでしょう。

## 高度な投球戦略：「ディングス」メソッド
主にバッティングのテクニックですが、「ディングスバント」（バントの構えをして内野手を前に引き出す）は投手に反応を強います。投手としてスタミナが低い場合は、これらの揺さぶりに乗らないようにしましょう。ストライクゾーンの四隅を突くことに集中してください。疲れた投手でも、ボールがストライクゾーンの端に完璧に配置されれば、たまにフライアウトを取れることがありますが、許容されるミスは極めてわずかです。

## FAQ

**Q: 自分のチームが得点すれば、スーパースターベースボールの投球スタミナは回復しますか？**

A: いいえ。このゲームのスタミナは一方通行です。スター投球や自責点によってポイントが失われると、その試合中にその投手のスタミナが回復することはありません。

**Q: 画面上に数値化されたスタミナバーは表示されますか？**

A: **スーパースターベースボールの投球スタミナ**を示す数値バーは表示されません。自責点の数とその投手が投げたスター投球の数を数えて、手動でポイントを把握する必要があります。

**Q: デッドボールを与えるとスタミナは減りますか？**

A: いいえ、デッドボール自体で直接スタミナポイントが減ることはありません。ただし、そのランナーが最終的に自責点として生還した場合は、その時点でポイントを失います。

**Q: 疲れた状態で投球するメリットはありますか？**

A: 統計的なメリットはありません。一部のプレイヤーは、相手を遅くて真っ直ぐなボールに誘い込む「餌」として利用することがありますが、経験豊富なプレイヤー相手には、ほぼ間違いなくさらなる失点を招く結果となります。
